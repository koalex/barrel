# Barrel

### Cборка и запуск

Для запуска проекта в системе должны быть глобально установлены зависимости:

- yarn

- pm2

```bash
# Установить yarn зависимости
yarn install
```

```bash
# Запуск сервера в production режиме
yarn start

# Запуск dev-сервера (режим разработки)
yarn run dev

# Deploy
yarn run deploy
```
